# SensorMonitor

PowerToys Command Palette 的 Dock 常驻硬件传感器显示扩展 —— 在屏幕边缘的 Dock 工具条上实时显示 CPU 频率、主板温度、GPU 温度。传感器数据来自开源的 [LibreHardwareMonitorLib](https://github.com/LibreHardwareMonitor/LibreHardwareMonitor)（MPL-2.0），由随扩展自动拉起的提权后台进程提供。

**状态：规划阶段。** 设计与实施计划见 [docs/](docs/)，Agent/开发者入口见 [CLAUDE.md](CLAUDE.md)。
